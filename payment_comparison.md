# Comparativa de Pasarelas de Pago para Argentina (2025)

Aquí tienes un análisis de las mejores opciones para vender software al mundo desde Buenos Aires.

| Característica | **LemonSqueezy** (Recomendado) | **Gumroad** | **Paddle** | **Mercado Pago** |
| :--- | :--- | :--- | :--- | :--- |
| **Tipo de Venta** | Global (Merchant of Record) | Global (Merchant of Record) | Global (Merchant of Record) | Local (Argentina) |
| **Retiro de Dinero** | **Transferencia Bancaria** (vía intermediarios) o **PayPal** | **Cuenta Bancaria Directa** | **Payoneer**, Wire Transfer | Banco Local |
| **Moneda de Cobro** | USD | USD | USD, EUR, etc. | Pesos Argentinos (ARS) |
| **Gestión de Impuestos** | Automática (Ellos se encargan) | Automática | Automática | Manual (Factura Electrónica) |
| **Ventaja Principal** | API moderna, manejo de licencias nativo | Muy fácil de empezar, marketplace propio | Muy robusto para B2B | Líder en Argentina |
| **Desventaja Principal** | Validación de cuenta puede demorar | Comisiones más altas en plan gratuito | Validación estricta | Solo cobras en ARS (inflación) |

## Análisis Detallado para tu Caso

### 1. LemonSqueezy (La opción equilibrada)

* **Por qué sí:** Tiene un sistema de **"License Keys"** nativo (vital para tu app). Te ahorra programar un generador de claves.
* **El dinero:** Te pagan vía transferencia bancaria internacional o PayPal. Muchos argentinos usan **PayPal -> Cuevas/Exchangers** o servicios como **Takenos/Bitwage** para traerlo a valor "blue".

### 2. Gumroad (La opción más fácil)

* **Por qué sí:** Es extremadamente fácil de configurar. Tienen "Gumroad License Keys" también.
* **El dinero:** Soportan **Depósito directo a cuenta bancaria en Argentina** (pero ojo con la pesificación oficial del banco).

### 3. Paddle (La opción "Empresa Grande")

* **Por qué sí:** Es el estándar de oro para SaaS B2B.
* **El dinero:** Pagan directo a **Payoneer**. Si ya usas Payoneer, es la mejor opción.
* **Contra:** Son muy exigentes para aprobarte la cuenta (te piden web corporativa, términos y condiciones legales, etc.).

### 4. Mercado Pago (La opción "Solo Argentina")

* **Por qué sí:** Todos en Argentina lo tienen.
* **Contra:** Si vendes a alguien de España o México, no pueden pagarte fácil. **Cobras en Pesos**.

## Mi Recomendación

**Sigue con LemonSqueezy** o cámbiate a **Gumroad** si quieres algo más "informal" al inicio.

* Si tienes cuenta en **Payoneer**, **Paddle** es excelente.
* Si quieres cobrar "en mano" en Argentina, usa LemonSqueezy -> PayPal -> Exchanger.
